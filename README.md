# Conversor de Divisas Inteligente con IA

Una aplicación web moderna y completa para la conversión de divisas con tasas de cambio en tiempo real, asistente inteligente para consejos sobre cuándo cambiar, visualización de tendencias históricas y alertas personalizadas.

## 🌟 Características Principales

### ✅ Funcionalidades Implementadas

#### 💱 Conversión de Divisas
- **Tasas de cambio actualizadas**: Datos en tiempo real (simulados con variaciones realistas)
- **10 divisas principales**: USD, EUR, GBP, JPY, CHF, CAD, AUD, CNY, MXN, COP
- **Conversión instantánea**: Resultados automáticos al cambiar cantidades o divisas
- **Cálculo preciso**: 4 decimales de precisión en las tasas

#### 🤖 Asistente IA Inteligente
- **Análisis de mercado**: Evaluación de volatilidad, tendencias y eventos económicos
- **Recomendaciones personalizadas**: Consejos basados en pares de divisas específicos
- **Timing óptimo**: Sugerencias sobre cuándo cambiar según el horario del mercado
- **Análisis técnico**: Niveles de soporte y resistencia
- **Eventos económicos**: Alertas sobre reuniones de bancos centrales y datos importantes

#### 📊 Visualización de Datos
- **Gráfico de tendencias**: Historial de 7 días con líneas suaves
- **Indicadores técnicos**: Media móvil simple y niveles clave
- **Controles interactivos**: Botones para cambiar períodos (7d, 30d, 90d)
- **Exportación de datos**: Posibilidad de descargar datos históricos

#### 🔔 Sistema de Alertas
- **Alertas personalizadas**: Crear alertas para tasas objetivo específicas
- **Gestión de volatilidad**: Notificaciones cuando el mercado es muy volátil
- **Recordatorios inteligentes**: Sugerencias basadas en condiciones del mercado

## 📖 Cómo Usar

### 1. Conversión Básica
1. Selecciona la divisa de origen en el campo "De"
2. Selecciona la divisa de destino en el campo "A"
3. Ingresa la cantidad a convertir
4. El resultado aparecerá automáticamente

### 2. Asesor IA
1. Haz clic en "Obtener Consejo" para recibir análisis personalizado
2. El asesor analizará:
   - Volatilidad actual del mercado
   - Tendencias técnicas
   - Eventos económicos próximos
   - Mejores horarios para cambiar
   - Niveles de soporte/resistencia

## 🔧 Configuración para Desarrollo

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a internet para cargar recursos externos

### Instalación Local
1. Clona o descarga los archivos del proyecto
2. Abre `index.html` en tu navegador
3. ¡Listo para usar!

### Personalización de la API
Para usar datos reales de divisas:
1. Regístrate en [ExchangeRate-API](https://www.exchangerate-api.com/)
2. Obtén tu clave API gratuita
3. Reemplaza `demo-key` en `js/currency-converter.js` con tu clave real

### Configuración de Monetización
El proyecto incluye espacios publicitarios listos para Google AdSense:

1. **Regístrate en Google AdSense**: https://www.google.com/adsense
2. **Obtén tu ID de editor**: `ca-pub-XXXXXXXXXXXXXXXX`
3. **Reemplaza en el código**:
   - `index.html`: Actualiza los IDs de anuncios
   - `js/ad-manager.js`: Configura tu ID de cliente
4. **Crea espacios publicitarios**: 728x90, 160x600, responsive
5. **Activa la monetización**: Los anuncios se cargarán automáticamente

El sistema detecta automáticamente AdBlock y muestra mensajes amigables para mantener la experiencia del usuario.
